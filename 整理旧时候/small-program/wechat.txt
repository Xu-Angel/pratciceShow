一、小程序文件结构
app(在根目录仅有一个)
    app.js(必填)
    app.json(必填)
    app.wxss
page(四个文件必须具有相同的路径和文件名)
    index.js(必填)
    index.wxml(必填)
    index.json
    index.wxss

二、配置
app.json
{
    "pages":[多个],
    "window":[
        "navigationBarTitle":"中巴巴变"
    ],
    "positon":"top",
    "tabBar":[
        {
            "pagePath":"",
            "text":""
            "icon":""
        }
    ]
}


page.json
{
    "navigationBarTitle":"中巴巴爸爸"
}

三、逻辑层
1、App() 用来注册一个小程序    getApp():全局函数，可以获取小程序的实例
2、Page() 用来注册一个页面

四、路由方式
wx.navigateTo() 跳转到非tabBar路径，保存原来页面
wx.redirectTo() 跳转到非tabBar路径，关闭原来页面
wx.switchTab() 跳转到tabBar路径


五、视图层
数据绑定  {{message}}
列表渲染(for循环)  wx:for="{{array}}"  
    item:遍历到的那一项 
    index:遍历到的索引
条件渲染  wx:if="{{}}"  wx:elif="{{}}"  wx:else="{{}}"   
模板
    定义模板   <template name="tmp"></template>
    引用模板   
        import "template.wxml"
        <template is="tmp"></template>

事件 bindtap 轻触
生命周期  
    页面加载  页面显示  页面隐藏  页面写在  下拉刷新  上拉触底  分享

组件  view  div       text span    swiper(swiper-item)
布局方法  flex
